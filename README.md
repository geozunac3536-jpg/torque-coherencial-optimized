## 📊 KPIs y predicciones falsables

- Índice de locking (LI ≥ 0.90)
- Correlación señal-referencia (R > 0.95)
- RMSESL < 0.10
- Estabilidad fraccional Øy(f*) < 3 × 10⁻¹⁶
- Cotas Yukawa sin violar PPN/WEP
